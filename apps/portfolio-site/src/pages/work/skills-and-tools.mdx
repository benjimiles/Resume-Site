I embarked on my journey in web development approximately **{new Date().getFullYear() - 2018} years ago**. Along the way, I delved into various programming languages and technology stacks, exploring both Back-End and Front-End domains.

Despite the wide-ranging field of web development, my passion and expertise lie in **Full-Stack Web Development** and **Mobile App Development**. These areas have captivated my interest, and I've dedicated my focus to mastering them over the past few years.

## Full-Stack Developer

I love creating something that is clean and minimalistic, attractive and has value, and of course, easy to use.

I really enjoy creating websites with **rich UI components**, including:

- **Web application**,
- **Documentation pages**,
- **CMS contents layout**,
- **Dashboard layout**,
- **E-commerce Stores**,
- and others.

But I still like to make simple website pages like **landing pages**. So, what tools did I feel comfortable using during the website creation?

### TypeScript

To start, let's talk about the programming language. **TypeScript** is my go-to choice, although website development can also be achieved using PHP, Python, and various other languages.

I've been working with TypeScript, a superset of JavaScript, for approximately **{new Date().getFullYear() - 2018} years**, and it has been a constant companion in my programming journey.

### React

Creating single-page application websites is something I truly relish, and React has been my preferred library for this purpose.

I've been utilizing React since 2020, which adds up to **{new Date().getFullYear() - 2020} years** of experience, allowing me to attain a high level of proficiency with the framework.

### Tailwind CSS

For now, I prefer using Tailwind CSS for styling, even though I have a longer history of {new Date().getFullYear() - 2019} years with SCSS.

I am not a newcomer to Tailwind CSS; in fact, I am quite **proficient** with it and find it to be an excellent tool for my projects.

### Framer Motion

When it comes to animations, I find Framer Motion to be my preferred choice. Making animations with JavaScript has never been as effortless as it is with Framer Motion, making the process incredibly smooth and intuitive.

### Next.js

I've been familiar with CRA (create react app) for a few years, but after discovering Next.js, I'm convinced it's the ideal web framework for me. **Its capabilities are simply unmatched!**
